Este es un ejemplo de cómo mostrar un número decimal en HTML:

3.14159

La constante π es aproximadamente igual a 3.14159.

En este ejemplo, hemos creado una clase CSS llamada "numero-decimal" para aplicar estilos a nuestro número decimal, como el tamaño de fuente y el color. Luego, utilizamos esta clase para dar formato a un número decimal y mostrarlo en la página web. Recuerda que si deseas realizar operaciones matemáticas o cálculos con números decimales en una página web, necesitarás JavaScript para lograrlo, ya que HTML y CSS por sí solos no son lenguajes de programación funcionales en ese sentido.
